## 🚀 How to Use

### Master Prompts
1. Open any `.md` file from the `master_prompt/` folder
2. Copy the full prompt content
3. Paste it into your preferred LLM (ChatGPT, Claude, Gemini, etc.)
4. Add your specific context or question after the prompt

### System Prompts
1. Open the relevant tool folder under `system_prompts/`
2. Reference the prompt to understand how the tool is configured
3. Use as inspiration to build your own custom agents or replicate behavior
